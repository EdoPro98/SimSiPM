\doxysubsection{Detailed Description}
Class storing informations relative to a single Si\+PM hitted cell. 

This class is used mainly to store informations relative to a single hit on a Si\+PM cell. Informations stored in thiss class will be used to generate the signal for each Si\+PM cell.
