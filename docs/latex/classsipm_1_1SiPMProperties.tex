Class storing all the parameters that describe a Si\+PM. 

\mbox{\hyperlink{SiPMDigitalSignal_8h_source}{Si\+PMDigital\+Signal.\+h}}

This class stores all the parameters and values used to describe a Si\+PM sensor or signal. It also allows to switch on or off some noise effects and can set different levels of detail in the evaluation of PDE.
