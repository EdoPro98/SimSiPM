\doxysubsection{Detailed Description}
Class that simulates the operation of an ADC converter. 

This class is used to convert a \mbox{\hyperlink{classsipm_1_1SiPMAnalogSignal}{Si\+PMAnalog\+Signal}} into a \mbox{\hyperlink{classsipm_1_1SiPMDigitalSignal}{Si\+PMDigital\+Signal}}. The signal is quantized using a given number of bits after it is amplified using a given gain.
